Box Sizing

Every element has the size of the content, padding, border, margin.

Static Positioning

Follows the normal flow of the document.

So that's static positioning.

Relative Positioning

Takes up it's normal space in the normal flow of the document even though it is offset from static positioning.

So that's relative positioning.

Absolute Positioning

Absolute positioning you will normally make sure it's parent is relative,
otherwise you are potitioning absolute so from the top-left corner of the page.
Notice, there is no space in the normal flow of the document reserved for an absolute element.
